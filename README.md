# Amazon-Sales-Data-Analysis-Excel 
This project showcases end-to-end data analysis skills using Microsoft Excel, focusing on cleaning, transforming, and visualizing a real-world Amazon sales dataset. The goal was to derive actionable business insights, identify top-performing products, and analyze pricing strategies.
